About My Work

TMSR Core Modeling using MCNPX for different Enrichment and Slab Spacing Variations

Description

This work is my Nuclear System Design coursework at Universitas Gadjah Mada. The TMSR-500 reactor (Thorium Molten Salt Reactor with a capacity of 500 MWe) is a nuclear reactor designed to be built in Indonesia which is a scale-up of the Molten Salt Reactor Experiment (MSRE) at the Oak Ridge National Laboratory, United States. In this work, simulations on the TMSR reactor core model were performed using MCNPX (Monte Carlo N-Particle Extended) software with graphite slab spacing and U-235 enrichment variations to determine the effect of graphite slab spacing and U-235 enrichment on the effective multiplication factor (keff) value.
